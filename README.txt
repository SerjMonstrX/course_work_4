# Курсовая №4 "Курсовой проект по курсу «ООП»"

ПО разработано в учебных целях, проект помогает осуществить поиск вакансий через API с ресурсов Headhunter и Superjob.

Для управления зависимостями в проекте использован Poetry.
Для работы с Superjob необходимо задать ключ API в файле 'src/variables.py'.

Git link "https://github.com/SerjMonstrX/course_work_4"
